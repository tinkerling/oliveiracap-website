# oliveiracap-website
Oliveira Capital — Website
Static one-page website for oliveiracap.com.

Structure
Single-file site — everything (HTML, CSS, JS, and logo assets) is contained in index.html.

Hosting
Published via GitHub Pages with a custom domain pointing to oliveiracap.com.

Updating
Edit index.html directly in GitHub (click the file → pencil icon → commit changes) or clone the repo locally and push changes. GitHub Pages will redeploy automatically within a few minutes.
